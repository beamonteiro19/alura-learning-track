# Resumo do Curso: Python - Crie sua Primeira Aplicação

## O que foi feito
- Desenvolvimento de um **aplicativo de gerenciamento de restaurantes** em Python.
- Implementação de um **menu interativo** com opções:
  - Cadastrar restaurante
  - Listar restaurantes
  - Alterar status (ativado/desativado)
  - Finalizar aplicação

## Aprendizados principais
- Uso de **listas** e **dicionários** para armazenar dados estruturados (nome, categoria, status).
- Criação de **funções** para modularizar o código (exibir menus, cadastrar, listar, alterar status).
- Aplicação de **docstrings** para documentar funções.
- Uso de **condicionais** e **operador ternário** para simplificar lógica.
- Manipulação de **strings** com métodos como `.ljust()` para alinhar colunas.
- Estrutura de controle com **try/except** para validar entradas do usuário.
- Uso de `os.system('clear')` para limpar o terminal e melhorar a experiência do usuário.

## Pontos importantes
- **Boas práticas**: separar responsabilidades em funções, documentar com docstrings, validar entradas.
- **Interatividade**: aplicação recebe inputs do usuário e responde dinamicamente.
- **Organização**: uso de menus e subtítulos para guiar a navegação.
- **Conceito chave**: dicionários permitem representar objetos com múltiplas propriedades (chave-valor).
